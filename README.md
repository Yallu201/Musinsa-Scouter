# [무신사스카우터](https://mscouter.web.app/)

## 개요

인터넷 쇼핑몰 [무신사](https://www.musinsa.com/)의 상품가격을 추적하여 보여주는 사이트.  
[https://mscouter.web.app/](https://mscouter.web.app/)

## 화면구성

#### 1. 메인페이지

- 검색바
- 일일랭킹 1000위 상품 목록

#### 2. 제품상세페이지

- 상품 이미지
- 당일 순위 및 가격
- 그래프(순위,가격)
- 그래프 조작 버튼
